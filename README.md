# Динамический UI-Генератор v2

Этот проект предоставляет набор JavaScript-классов для динамической генерации пользовательского интерфейса на основе JSON-схемы. Он подходит для веб-приложений, где структура UI определяется на сервере.

## Возможности

- Генерация UI по JSON-схеме.
- Поддержка вкладок, групп, параметров (readonly, number, flags) и команд.
- Управление видимостью и доступностью элементов.
- Интеграция с WebSocket для обмена данными.
- Централизованное событийное взаимодействие.
- Логирование с настраиваемым уровнем.
- Валидация схемы.
- Предотвращение утечек памяти с помощью метода `destroy`.

## Структура

- `src/`: Исходный код основных классов.
- `example/`: Пример HTML-страницы и использования.
- `docs/`: Документация.
